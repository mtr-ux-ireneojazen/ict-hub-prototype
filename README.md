# ICT Hub

A prototype website developed as a proposed platform for the Senior High School ICT Strand Representative.

## Purpose

ICT Hub aims to centralize:
- Announcements
- Events
- Lost & Found
- Student Concerns
- Learning Resources
- Student Project Showcase

The project is intended to improve communication and organization within the ICT strand while remaining simple, accessible, and easy for future student officers to maintain.

> This repository contains a prototype and is intended for demonstration purposes.
